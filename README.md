
# React boilerplate with Hot Loader plugin
A boilerplate React project, setup to work for scss, with bootstrap included and Hot Loader plugin. 

# To get started do:
1. git clone https://github.com/cristiAndreiTarasi/React_Hot_Loader_App_Boilerplate.git
2. cd React_Hot_Loader_App_Boilerplate
3. npm install
4. npm start - to start the development server and the hot load
    or
   npm run build - to start the production build
